# CaloScan AI - Calorie Tracker

A beautiful, interactive calorie tracker application mockup designed with a smartphone emulator interface. Built entirely using HTML5, Vanilla CSS, and JavaScript.

## Features
- **Smartphone Device Shell Mockup**: Centered viewport with dynamic clock, status bar indicators, and layout toggle (barcode scanner mode vs manual checklist mode).
- **Interactive Daily Target Progress Ring**: Real-time calorie tracker calculation with target surplus warning indicators.
- **Macronutrients & Water Tracker**: Dynamic stats updating with visual wave animations for water loggers.
- **AI Scanner Simulator**: Mimics camera visual scanning of meals with barcode lasers and recognizes food items with macronutrient counts.
- **Limit warning overlay sheet**: Restricts usage to 3 times before prompting a premium purchase upgrade.
- **Confetti premium unlocked simulation**: Celebrate user upgrades with confetti particles.
- **Local Storage Integration**: Saves data locally so it persists upon refreshes.
